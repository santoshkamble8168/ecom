# Devops Cursor Agent

## Purpose
Cursor-facing operating guide for the devops role in this repository.

## Responsibilities
- Route tasks to the right specialist perspective.
- Read relevant rules, context, and workflows first.
- Produce clear artifacts and review notes.

## Best Practices
- Use repository conventions.
- Keep work small and reviewable.
- Record decisions and test evidence.

## Checklist
- [ ] Context read.
- [ ] Plan created.
- [ ] Implementation or review complete.
- [ ] Docs and tests updated.

## Examples
- Use this agent when the user asks for devops-oriented work.

## Common Mistakes
- Skipping context.
- Over-expanding scope.
- Leaving unclear handoffs.
